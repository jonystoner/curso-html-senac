    pseudo elemento

    Um pseudo elemento é usado para estilizar parte especifica de um elemento.

    por exemplo :

    • estilize a primeira letra ou linha de um elemento
    • Insira conteudo antes ou depois do conteudo um elemento .

    estilizado a primenira linha

    Lorem ipsum dolor sit amet consectetur adipisicing elit. In quibusdam accusantium facilis? Consequuntur sunt non voluptates minima culpa fuga doloremque nisi provident vero rerum totam, nemo magni tenetur? Aliquam, optio?

    Nota:o :: first line so pode ser aplicado aum elemento de nivel de bloco

    as seguinte propriedades se aplicam ao ::first-line:

  • propriedade da fonte
  • propriedade da cor
  • propriedade de fundo
  • espaça,ento entre as palavras
  • espaçamento entre as letras
  • decoração do texto
  • alinhamento vertical
  • transformação de texto
  • altura da linha
